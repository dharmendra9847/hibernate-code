# Quiz App (Java + Maven)

A simple console-based Quiz Application built using Java and Maven.
This project demonstrates basic object-oriented design and service-based structure.

---

## 🚀 Features

* Create and manage quiz questions
* Simple question service layer
* Console-based interaction
* Clean modular structure

---

## 🛠️ Tech Stack

* Java
* Maven
* OOP (Object-Oriented Programming)

---

## 📂 Project Structure

```
src/main/java/com/quiz/myapp/
│── Main.java
└── services/
    │── Question.java
    └── QuestionService.java
```

---

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/quiz-app.git
cd quiz-app
```

### 2. Build the project

```bash
mvn clean install
```

### 3. Run the application

```bash
mvn exec:java -Dexec.mainClass="com.quiz.myapp.Main"
```
